Tim Berners-Lee

(Londres, 1995). Científico de la computación británico. Habilitó un sistema para poder ver páginas web o documentos de hipertexto por medio de Internet. También se desempeña como director del World Wide Web Consorcio (W3C), que supervisa los estándares para Internet y la World Wide Web. Tim Berners-Lee también se preocupa por cuestiones relacionadas con la libertad de información y la censura que se da en Internet.

Desarrolló lo que por sus siglas en inglés se denomina: HTML o Lenguaje de Etiquetas de Hipertexto; el protocolo HTTP (HyperText Transfer Protocol), y el sistema de localización de objetos en la web URL (Uniform Resource Locator).

Es importante porque fue la primera persona que logró establecer la comunicación entre un servidor y un cliente utilizando el protocolo HTTP, supervisó y estandarizó el desarrollo de las tecnologías necesarias sobre las cuales se encuentra fundada la Web y además que permiten el adecuado funcionamiento del internet, además de la creación del lenguaje HTML.
